Magnum Solutions CPAN RPM Repository

The Magnum Solutions CPAN RPM repository has closed down.

While we still believe that if you're installing CPAN modules to be used by the system Perl, it's a good to use a package manager, we do not think that you should be using the system Perl to run your own applications. Instead, we recommend using a tool like perlbrew to install your own Perl. Alternatively, you could use a containerisation tool like Docker to isolate your application from the underlying system.

If you need help setting up these alternatives (or building your own RPMs from CPAN modules), please get in touch with us at hello@davecross.co.uk.
